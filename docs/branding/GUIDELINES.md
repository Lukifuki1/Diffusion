# DiffusionHub Enterprise - Brand Guidelines

## 🎨 Vizualna Identiteta

### Barvna Shema

**Primary Colors:**
- **Enterprise Blue**: `#1E40AF` (Main brand color)
- **Deep Purple**: `#7C3AED` (Accent color)
- **Dark Navy**: `#0F172A` (Text and backgrounds)

**Secondary Colors:**
- **Light Gray**: `#F3F4F6` (Backgrounds)
- **Success Green**: `#10B981` (Status indicators)
- **Warning Orange**: `#F59E0B` (Warnings)
- **Error Red**: `#EF4444` (Errors)

**Gradient:**
```css
background: linear-gradient(135deg, #1E40AF 0%, #7C3AED 100%);
```

### Tipografija

**Primary Font:** Inter (Clean, modern, professional)
- Regular: 400
- Medium: 500
- Bold: 700

**Secondary Font:** JetBrains Mono (Code blocks)
- For technical documentation and code samples

### Logo Usage

**Clear Space:** Minimum padding equal to the height of the logo's main element
**Minimum Size:** 32px height for digital, 1 inch for print
**Backgrounds:** Use on light or dark backgrounds with sufficient contrast

## 📊 Presentation Template

**Slide Structure:**
1. Title Slide (Brand gradient background)
2. Problem Statement
3. Solution Overview
4. Technical Architecture
5. Pricing & Plans
6. Call to Action

**Color Usage in Presentations:**
- Primary: Brand blue/purple gradient
- Secondary: Light gray backgrounds
- Accent: Success green for positive metrics

## 📝 Tone of Voice

**Professional yet approachable**
- Clear, concise language
- Technical accuracy without jargon overload
- Focus on enterprise benefits (scalability, reliability, support)

**Key Messages:**
- "Enterprise-grade reliability"
- "Scalable to 100+ concurrent generations"
- "99.9% uptime SLA"
- "Professional support included"

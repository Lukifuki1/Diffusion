# AuraFlow Studio Frontend Assets

This directory contains frontend assets for AuraFlow Studio.

## Note
The primary frontend for AuraFlow Studio is now the Blazor Server UI integrated in the .NET application (`/src/AuraFlow/`). This directory contains legacy/styling assets that can be used for reference or future frontend development.

## Files Included:

- `branding/custom.css` - AuraFlow color scheme (#1E40AF, #7C3AED)
- `assets/logo-auraflow.svg` - Custom logo
- `assets/auraflow-theme.js` - Theme configuration
